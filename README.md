 Interactive Quiz & Game Hub

A creative, fun, and interactive web app built with HTML, CSS, and JavaScript.  
This project is a multi-round quiz and game hub, combining knowledge tests with mini-games for a full gamified experience.  

---

 Features

- Multi-Round Quiz: 
  - 5–10 rounds with easy, medium, and hard questions.  
  - Questions are randomized each game for replayability.  
  - Immediate feedback on answers.  

- Gamified Mini-Games:  
  - Rock-Paper-Scissors  
  - Heads or Tails (Coin Flip)  
  - Additional mini-games added for fun and challenge.  
  - Each game contributes to your overall score.  

- Progressive Unlocking:
  - Win a quiz round → unlock next mini-game.  
  - Complete all games → see your final score and winner announcement.  

- Interactive UI:  
  - Eye-catching, animated buttons and transitions.  
  - Hover effects, confetti effects for wins.  
  - Light and playful color scheme with responsive design.  

- Replayable & Randomized:  
  - Games loop randomly for endless fun.  
  - Users can replay quizzes and mini-games multiple times.  

---
 How to Play

1. Click Start Game to begin the quiz.  
2. Answer each quiz question to progress.  
3. Complete mini-games like Rock-Paper-Scissors and Heads/Tails.  
4. After completing all rounds, your winner score is displayed.  
5. Replay anytime to improve your score and enjoy randomized rounds.
